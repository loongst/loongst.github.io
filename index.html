日常随笔-vim安装与配置

# ubuntu14.04安装vim
1.apt-get install vim   #安装目录默认为/usr/share/vim 
2.安装ctags 下载地址http://ctags.sourceforge.net;  
  tar -zxvf ctags.xxx.gz; 
  ./configure;
  make;
  make install; 
 在编写源代码的目录下执行：ctags -R 生成ctags；在vimrc文件中添加set tags=/源码路径/tags；使用ctrl+]查看函数定义
3.安装TagList 下载地址http://www.vim.org/scripts/script.php?script_id=273 
 解压到~/.vim文件夹中，在vimrc中添let Tlist_Show_One_File=1换行let Tlist_Exit_OnlyWindow=1 ；
 在使用vim中，底行命令输入Tlist
4.安装WinManager 下载地址http://www.vim.org/scripts/script.php?script_id=95 
 解压到~/.vim文件夹中，在vimrc中添加let g:winManagerWindowLayout='FileExplorer|TagList' 换行nmap wm :WMToggle<cr>
 一般模式下输入wm使用（注意：不是底行命令模式）
